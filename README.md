# Digital Marketplace for Fish Waste Recycle System

## Overview

Digital Marketplace for Fish Waste Recycle System is a web-based platform that connects fish waste sellers with recycling companies. The system helps manage fish waste collection, requirements, orders, and transactions efficiently.

## Technologies Used

### Frontend

* React.js
* JavaScript
* HTML5
* CSS3

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* REST APIs

### Database

* MySQL

## Features

### Admin

* Admin Login
* Manage Sellers
* Manage Companies
* Monitor Orders
* System Management

### Seller

* Register and Login
* Add Fish Waste Details
* Manage Waste Inventory
* View Orders

### Company

* Register and Login
* Post Waste Requirements
* Browse Available Fish Waste
* Place Orders

### Security

* JWT Based Authentication
* Role Based Access Control
* Secure API Access

## Installation

### Backend

```bash
mvn clean install
mvn spring-boot:run
```

### Frontend

```bash
npm install
npm start
```
